## ğŸ†• For New Users (First Time Setup)

### Step 1: Sign Up / Sign In
- Go to the sign-in page
- Create a new account or sign in with existing credentials
- If signing up, verify your email first

### Step 2: Welcome Screen
- After signing in, you'll see a **Welcome Intro** screen
- This explains what Dearly is about
- Click **"Let's Begin âœ¨"** to continue

### Step 3: Receiver Setup
- You'll see the **Receiver Name Setup** screen
- Enter:
  - **Receiver Name**: The name of the person you're writing to (e.g., "Faith", "John", etc.)

- After saving receiver information, you'll be redirected to the **Dashboard**
- You can now:
  - Create letters (My Letters)
  - Plan dates (Date Planning)
  - Create games (Games)

## â“ Common Questions

### Q: Do I need to set up receiver information every time?
**A:** No! You only set it up once during initial setup. After that, you can update it anytime via the Profile modal.

### Q: Can I change the receiver later?
**A:** Yes! Click the profile icon (ğŸ‘¤) in the Dashboard header, then edit the receiver information.

### Q: What if I skip the receiver setup?
**A:** You can't skip it - it's required to create letters. The app will guide you through it on first login.

### Q: Can I have multiple receivers?
**A:** Currently, the app supports one receiver per sender account. To write to a different person, you would need to update the receiver information in your profile.
